## Iniciar Projeto

```cl
# Clonar o repositório
git clone https://github.com/animoConsultoriaTi/consultoriaVendetudo360.git

# Entrar no diretório
cd consultoriaVendetudo360

# Abrir VSCode
code .

# Instalar Dependências
npm install
```

## Editar Projeto

```cl
# Desenvolvimento
npm run dev

# Build Final
npm run build
```

git checkout <suaBranch>
